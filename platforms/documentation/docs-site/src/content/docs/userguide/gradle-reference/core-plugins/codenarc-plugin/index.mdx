The CodeNarc plugin performs quality checks on your project&#8217;s Groovy source files using [CodeNarc](https://codenarc.org/) and generates reports from these checks.

## Usage

To use the CodeNarc plugin, include the following in your build script:

{/* converter-gap: node=sample dir="snippets/reference/core-plugins/codeQuality/kotlin" files="build.gradle.kts[tags=use-codenarc-plugin]" */}

{/* converter-gap: node=sample dir="snippets/reference/core-plugins/codeQuality/groovy" files="build.gradle[tags=use-codenarc-plugin]" */}

The plugin adds a number of tasks to the project that perform the quality checks when used with the [Groovy Plugin](groovy_plugin.mdx#groovy_plugin). You can execute the checks by running `gradle check`.

## Project layout

The CodeNarc plugin expects the following project layout:

```text
<root>
└── config
    └── codenarc           // <1>
        └── codenarc.xml   // <2>
```
