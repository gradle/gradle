The [Signing Plugin](signing_plugin.mdx#signing_plugin) can be used to sign all artifacts and metadata files that make up a publication, including Maven POM files and Ivy module descriptors. In order to use it:

1. Apply the Signing Plugin
2. Configure the [signatory credentials](signing_plugin.mdx#sec:signatory_credentials) — follow the link to see how
3. Specify the publications you want signed
