The ANTLR plugin extends the Java plugin to add support for generating parsers using [ANTLR](http://www.antlr.org/).

:::note
The ANTLR plugin supports ANTLR version 2, 3 and 4.
:::

## Controlling the ANTLR generator process

The ANTLR tool is executed in a forked process. This allows fine grained control over memory settings for the ANTLR process. To set the heap size of an ANTLR process, the `maxHeapSize` property of [AntlrTask]({groovyDslPath}/org.gradle.api.plugins.antlr.AntlrTask.html) can be used. To pass additional command-line arguments, append to the `arguments` property of [AntlrTask]({groovyDslPath}/org.gradle.api.plugins.antlr.AntlrTask.html).
