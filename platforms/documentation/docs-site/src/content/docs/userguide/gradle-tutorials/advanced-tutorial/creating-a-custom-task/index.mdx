## Step 1: About Custom Tasks

If your plugin introduces new functionality, you’ll typically define a custom task type.
This allows you to encapsulate specific logic, like sending a Slack message, into a reusable and configurable Gradle task.
We&#8217;ll then register this task in our plugin so users can execute it with a simple command.
