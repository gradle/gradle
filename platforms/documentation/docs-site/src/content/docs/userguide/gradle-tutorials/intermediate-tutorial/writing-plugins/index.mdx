## Step 4. Run the custom Task

Finally, it&#8217;s time to run the new task.

First, let&#8217;s inspect some source code:

```java
package authoring.tutorial;

import com.gradle.CustomLib;

public class App {
    public String getGreeting() {
        return "CustomLib identifier is: " + CustomLib.identifier;
    }

    public static void main(String[] args) {
        System.out.println(new App().getGreeting());
    }
}
```

Next, let&#8217;s run the task with `./gradlew :app:license`:

```bash
$ ./gradlew :app:license
```
