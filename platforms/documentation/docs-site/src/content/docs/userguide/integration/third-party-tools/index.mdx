## Continuous integration

## How to integrate with Gradle

There are two main ways to integrate a tool with Gradle:

- The Gradle build uses the tool
- The tool executes the Gradle build

The former case is typically [implemented as a Gradle plugin](plugins.mdx#custom_plugins).
The latter can be accomplished by embedding Gradle through the [Tooling API](tooling_api.mdx#tooling_api).

## Test integration

If your platform uses an external testing suite, and you want to integrate with the [HTML Test Report](java_testing.mdx#test_reporting), you can use the [Test Reporting API](test_reporting_api.mdx#test_reporting_api).
